# Data Directory

This directory contains sample datasets for testing and demonstration purposes.

## Files

- `accident_50k.csv` - Sample dataset with 50,000 accident records
- `accident_sample_10.csv` - Minimal sample (10 records) for quick testing
- `accident_sample_15.csv` - Small sample (15 records)
- `accident_sample_100.csv` - Small sample (100 records)
- `accident_sample_500.csv` - Medium sample (500 records)
- `accident_sample_1000.csv` - Medium sample (1,000 records)
- `accident_sample_5000.csv` - Large sample (5,000 records)

## Full Dataset

The complete US Accidents dataset is available on Kaggle:
**[US Accidents (2016 - 2023)](https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents)**

### Download Instructions

1. Visit the Kaggle dataset page
2. Download the full dataset (approximately 7.7 million records)
3. Place it in this directory if you want to use it with the application

## Usage

These sample files can be used with the "Accident Dashboard" tab in the Streamlit application to explore visualizations and insights without downloading the full dataset.
